# Weather Diary

**Автор:** Александров Мирослав

## Краткое описание
Это приложение позволяет пользователю добавлять записи о погоде, хранить их в файле JSON, фильтровать по дате и температуре, а также просматривать и управлять всеми записями.

## Как запустить
1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте или склонируйте проект.
3. Запустите файл `weather_diary.py` двойным кликом или через командную строку:
```bash
python weather_diary.py
